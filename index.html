Welcome to MombasaJS

MombasaJS is a usergroup of Javascript enthusiasts in Mombasa, Kenya

Mission...

Organize fun and interactive meetups to share and discuss interesting topics related to Javascript.

...Our Goals

Create a network of Javascript developers in Mombasa

To update ourselves with the latest news related to Javascript

Build projects using Javascript that solve some of the problems in our society

Why Javascript?

Easy to learn.

Awesome tools and frameworks for your projects.

Such as..

  • NodeJS for networking
  • PDF.js for loading pdf files
  • Backbone and AngularJS for building large scale MV* applications
  • Grunt and Yeoman for automation
  • jQuery for giving your webpage a dynamic and interactive feel

Most open and popular programming language. My opinion based from GitHub's popular repositories. Most of them are Javascript based.

Meetup model

  • Meetups should contain atleast one demo of a Javascript framework/library in action.
  • Tips for writing better Javascript. Example: good coding practices, style etc.
  • Sharing of articles, resources and tutorials related to Javascript.
  • Question and answer session. Can't figure it out? Together we will find a solution..

{In a nutshell}

Don't leave the meetup if you haven't learnt something new in the universe of Javascript!
